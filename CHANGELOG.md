# Changelog

All notable changes to this repository will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-14

### Added
- Initial repository structure
- Comprehensive README.md with phenomenon documentation
- api.json following Wyrdness schema v1.0.0
- SOURCES.md bibliography
- Issue templates for sighting reports, corrections, and source additions
- Pull request template

### Data Included
- Basic phenomenon metadata and classification
- Physical characteristics and abilities
- Geographic distribution and hotspots (Khumbu, Menlung Basin)
- Historical timeline (1832-2017)
- 2 notable sightings (Tombazi 1925, Shipton footprints 1951)
- Evidence catalog (physical, photographic)
- 2017 DNA analysis results
- Theories (paranormal, scientific, skeptical)
- Cultural significance and Himalayan Indigenous perspectives
- Research organizations and notable researchers
